# Avatar - Example

Avatars are detailed portraits of the person you're talking to. The dashboard reads avatar files for the Reputation page and uses them as raw material for AI copy generation.

**This file is inactive.** The leading underscore in the filename (`_avatar-example.md`) tells the dashboard to skip it, so it doesn't count toward your avatar list or scoring. To activate this as your first real avatar, rename the file to remove the leading underscore (e.g. `avatar-jane.md`) and fill in the sections below with the actual person you serve. Build it from real conversations, calls, or DMs - the closer to verbatim, the more useful.
