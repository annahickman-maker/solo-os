# Master Todo

Your master todo list. Tasks live as individual files in `00_System/tasks/` (one file per task, frontmatter on top). This file is a human-readable index; the dashboard reads the tasks folder directly.

Sections the dashboard understands when grouping tasks:

- **This Week** - top-priority tasks for the current week
- **Next** - upcoming tasks
- **Someday** - parked
- **Recently Done** - completed in the last 2 weeks (kept for momentum)

Create new tasks from the dashboard's Focus page, or drop new files into `00_System/tasks/`.
