# YouTube Script Outline

Builds the structural skeleton of the video. Not the full script - the bones. Each section is bullet-pointed with the key beats and placeholders for stories. The user fills in the actual sentences using the section-specific skills (intro, context, value, cta, outro) or by writing themselves.

---

## Preflight

1. Read `04_YouTube/core_channel-positioning.md`. If missing, STOP: "Run /youtube-onboarding first to set up channel positioning."
2. Read `01_Core/core_voice-style.md`, `01_Core/core_audience.md`, `01_Core/core_positioning.md`, `01_Core/core_ip.md`, `01_Core/core_offer-suite.md`.
3. Check `05_Assets/` for relevant stories, proof points, and POVs.

---

## Required inputs

Confirm before building. If missing, ask.

- Finalised title (must be locked - if not, point user to /youtube-title)
- The transformation (before/after for the viewer)
- The baseline belief (what the viewer currently thinks)
- The contrarian flip (what the user actually believes)
- One strong proof point from the user's experience
- 3-5 value points with enough depth to outline (each: WHAT it is, WHY it matters, HOW to apply it)
- CTA (default to channel CTA from `core_channel-positioning.md` unless this video drives to a different one)
- Next video to link in the outro (search `05_Assets/Transcripts/` or ask the user)

If the user wants help gathering these, point them to /youtube-script (the orchestrator).

---

## Build the outline

Use the structure below. Each section is bullet-pointed. Do not write full sentences for the intro, value sections, CTA, or outro - those are handled by the section-specific skills. This is the skeleton only.

```
# [Working title]

**Transformation:** [before → after]
**Baseline belief:** [what the viewer thinks]
**Contrarian flip:** [what the user actually believes]
**Proof point:** [the specific result/story used as credibility]

---

## Intro (handled by /youtube-script-intro)

- Opens with: [opening pattern - personal admission / credibility statement / direct tension / concrete result / story-as-proof / direct context]
- Hits 3 Cs in first 30 seconds
- 5 parts: Context, Baseline Belief, Contrarian Flip, Proof, Plan

---

## Mindset shift / Context section (handled by /youtube-script-context)

- Mirror the viewer's current situation
- Name what's not working and why (logic of their current approach + why it fails)
- The AHA - the contrarian flip restated as a turning point
- What to do instead (briefly introduce the new approach)
- Why this works better (build the buy-in here)
- Bridge into value

**Story placeholder:** [if a personal story from `05_Assets/Stories/` fits the AHA, name it here]

---

## Value section 1 - [Point name]

- WHY: [why this matters - build curiosity, emphasise importance]
- WHAT: [the insight, tip, or step]
- HOW: [examples, stories, how to apply]
- PAYOFF + REHOOK: [satisfy the point + open curiosity gap into next section]

**Story placeholder:** [story that would strengthen this point, if any]

---

## Value section 2 - [Point name]

(same structure)

---

## Value section 3 - [Point name]

(same structure)

---

(continue for 4-5 total value sections)

---

## CTA (handled by /youtube-script-cta)

- Barrier: [what makes implementation hard]
- Shortcut: [the offer/freebie as the natural shortcut]
- Alignment: [how it connects to what was just taught]
- Casual invite: [where to get it]

CTA points to: [primary CTA from `core_channel-positioning.md` unless overridden]

---

## Outro (handled by /youtube-script-outro)

- Reinforce transformation (one line - what they just learned will give them)
- Curiosity gap (what's still missing from this video)
- Next video: [video title and reason it's the natural next step]
```

## Story check

Once the outline is laid out, scan for places a personal story or experience would strengthen the teaching:

> "Here are the spots where a personal story would make this land:
>
> - [Section]: A story about [topic] would work here - something that shows [what it demonstrates]
> - [Section]: A specific example or result here would make this feel real
>
> Do you have stories for any of these?"

Bank anything new the user shares to `05_Assets/Stories/`.

---

## Hand off

Once the outline is locked:

> "Outline is ready. Three options from here:
>
> 1. **Go script the whole thing** - run /youtube-script (orchestrator) and it'll walk you through each section
> 2. **Script just specific sections** - call /youtube-script-intro, /youtube-script-context, /youtube-script-value, /youtube-script-cta, or /youtube-script-outro for the part you want
> 3. **Go film with the outline as your guide** - use this as your speaking notes and come back if you want to script anything specific"

Save the outline to `04_YouTube/Scripts/[slug].md` (slug = 2-3 words from the title).

---

## Rules

- This skill produces the outline, not the script. Do not write the intro/sections word-for-word.
- Each section's bullet points are placeholders. The section skills fill them in.
- Always include story placeholders so they're top of mind during scripting.
- Never invent value points, proof, or stories.
- Never use - (em dash). Use - instead.
